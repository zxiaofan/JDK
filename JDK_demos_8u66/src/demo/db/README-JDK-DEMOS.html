Java™ DB demos

Java DB is Oracle's distribution of the open source Apache Derby Java database. It is distributed as part of the Java™ Development Kit.

Please refer to documentation here for details. The release notes show the version of the documentation applicable to the Java DB present in this kit.

This directory contains the original demos distributed with Derby. Note! Files in this directory contain paths which assume this directory is located in $DERBY_HOME/demo, i.e. $JAVA_HOME/db/demo. In the JDK, demos are optional and can be found located inside $JAVA_HOME/demo/db. Please allow for this difference when consulting the README files contained here.

The demos comprise example programs and databases demonstrating different aspects of the Apache Derby database system, as well as templates for custom configuration.

Please refer to "readme" or "example" files in the respective subdirectories for more information about each demo.
